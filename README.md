# Smart School Admin Portal

Next.js admin portal starter for your Smart School backend.

## Included
- login page
- protected admin layout
- sidebar + topbar
- dashboard
- notices page wired to backend
- students / parents / teachers / classes / attendance / fees / timetable / results / settings shells

## Setup
```bash
npm install
```

Create `.env.local`
```bash
NEXT_PUBLIC_API_BASE=https://school-backend-1-be49.onrender.com
```

Run
```bash
npm run dev
```
